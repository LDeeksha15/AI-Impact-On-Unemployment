# AI Impact on Unemployment in India

## Project Overview

This ETL and Data Analytics project analyzes unemployment trends in India using multiple datasets. The project studies state-level and national-level unemployment changes and evaluates the impact of AI-driven job opportunities on employment patterns.

---

## Objective

To analyze unemployment trends before and after AI/COVID, compare state-level and total-level unemployment rates, and understand AI-driven job creation in India.

---

## Datasets Used

- Unemployment in India dataset  
- Unemployment Rate up to 2020 dataset  
- Unemployment Rate Total dataset  
- AI Job Trends dataset  

---

## Tools Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## ETL Process

### Extract
Collected unemployment and AI job datasets from multiple sources.

### Transform
- Handled missing values  
- Removed duplicates  
- Standardized date formats  
- Created Year and Period columns  
- Classified data into Before AI and After AI  

### Load
Prepared cleaned datasets for analysis and visualization.

## Key Findings

- State-level unemployment increased from **9.4% → 15.1%**
- National-level unemployment increased from **6.6% → 9.8%**
- AI created approximately **66 million job opportunities**
- Job market is shifting due to AI transformation
- Reskilling is essential for future employment
